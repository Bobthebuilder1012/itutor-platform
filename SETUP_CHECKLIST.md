# iTutor Setup - Super Simple Checklist ✅

**Print this or keep it open while you setup!**

---

## Before You Start

**Get these from team lead:**
- [ ] GitHub invite (check email and accept)
- [ ] `.env.local` credentials (via secure message)

---

## Step 1: Install Software (15 minutes)

Just download and install these 3 things:

- [ ] **Cursor** → https://cursor.sh/
- [ ] **Node.js** → https://nodejs.org/ (LTS version)
- [ ] **Git** → https://git-scm.com/downloads

**Installation:** Use default settings for everything, just keep clicking "Next"

---

## Step 2: Clone the Project (2 minutes)

1. [ ] Open Cursor
2. [ ] Press `Ctrl+L` (opens AI chat)
3. [ ] Tell Cursor:

```
Clone this repository:
https://github.com/Bobthebuilder1012/itutor-platform.git

Then open it in Cursor
```

---

## Step 3: Let Cursor Do the Setup (5 minutes)

**Tell Cursor AI (one at a time):**

1. [ ] **"Install all npm dependencies"**
   - Wait 2-3 minutes

2. [ ] **"Switch to the dev branch"**

3. [ ] **"Create a .env.local file with this content:"**
   - Then paste the credentials team lead sent you

4. [ ] **"Start the development server"**

---

## Step 4: Test It Works (1 minute)

- [ ] Open browser: **http://localhost:3000**
- [ ] See iTutor homepage? **YOU'RE DONE!** 🎉

---

## Daily Workflow (Simple!)

### Starting Work:
**Tell Cursor:** "Pull latest from dev and start the server"

### Making Changes:
1. Edit files in Cursor
2. Save (Ctrl+S)
3. Browser auto-refreshes
4. Test it works

### Ending Work:
**Tell Cursor:** "Commit my changes with message: [what you did] and push to dev"

---

## One Golden Rule

**❌ NEVER push to main**  
**✅ ALWAYS work on dev**

That's literally the only rule. Follow this and you can't break anything!

---

## If Something Goes Wrong

**Tell Cursor:** "Something's broken, help me fix it"

Or ask your team lead!

---

**That's the whole setup. Seriously. Welcome to the team!** 🚀
